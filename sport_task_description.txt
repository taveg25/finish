


    На финиш прибывают по очереди спорсмены.
    Известно про каждого спортсмена:
        - фамилия;
        - номер;
        - результат.
    Количество спортсменов не ограничено.
    
    Требуется:
    
    1 вариант. После прибытия каждого спортсмена
        выводить список из пяти лучших на данный момент.
        Результаты спортсменов повторяться не могут.
        
    2 вариант. При появлении в числе лучших спортсмена 
        с повторяющимся результатом считать это ошибкой.

    3 вариант. Сделать ввод не с клавиатуры, а с помощью 
        видеокамеры, путем преобразования qr-кода в результат.
	повторяющийся результат ошибкой не считать, просто не записывать.
        
    Дополнительная информация:
    
    1. Данные вводятся с клавиатуры, каждый элемент
        данных вводится по отдельности.
        
    2. Смысл результата - время в секундах. Лучше, когда меньше.
    
    3. Лучших только 5, больше не требуется.
    
    Алгоритм  решения:
    
    1. Создаем список лучших (вначале пустой).
    ||:
    2. Вводим сведения о прибывшем спортсмене.
    3. Добавляем эти сведения в список.
    4. Сортируем список по ухудшению результата.
    5. Если элементов в списке больше пяти,
        удаляем последний.
    6. Выводим список.
    :||
